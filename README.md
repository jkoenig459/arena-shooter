# Arena Shooter

An **arena FPS prototype** built in **Unreal Engine**, focused on first-person combat against AI enemies that navigate the level using a NavMesh. Originally created for GAM-207.

## Overview

- **Engine:** Unreal Engine
- **Scripting:** Blueprints
- **Genre:** Arena first-person shooter
- **Status:** Prototype

## What I built

- First-person character controller (with double jump)
- Enemy AI with movement and shooting behavior
- Enemy pathfinding via NavMesh
- Health, damage, and respawn systems
- Score tracking and UI
- Health pickups
- Custom test level

## Skills demonstrated

Unreal Engine · game AI · NavMesh navigation · game physics · first-person controls · UI

## Planned improvements

- Power-ups and weapon pickups
- Improved NavMesh navigation and level design so enemies path more cleanly around ramps
